hugo+mac搭建博客并部署github

2019.05.18

1.mac环境安装hugo

安装步骤参考：

hugo官网-安装Hugo

我用的是Tarball方式:

在浏览器中打开https://github.com/gohugoio/hugo/releases。

通过向下滚动查找当前版本并查找“最新版本”的绿色标记。

下载适用于Mac的当前tarball。名称将是，版本号hugo_X.Y_osx-64bit.tgz在哪里X.YY。

默认情况下，tarball将保存到您的~/Downloads目录中。

在~/Downloads找到 hugo_0.55.5_macOS-64bit.tar.gz 解压为 hugo_0.55.5_macOS-64bit

将 hugo_0.55.5_macOS-64bit 复制到/usr/local/bin/目录中

打开终端输入 hugo version

  $ hugo version
  Hugo Static Site Generator v0.37.1 darwin/amd64 BuildDate

成功。

2.在本地搭建hugo博客

创建新站点:

$ hugo new site blog //*创建文件名为blog的新站点 （生成blog文件在/user/path/to/site下）
$ cd blog   //进入该文件
$ tree blog //显示blog目录结构
# 目录结构
#blog
#├── archetypes
#│   └── default.md
#├── config.toml //是配置文件，在里面可以定义博客地址、构建配置、标题、导航栏等等
#├── content //content 是博客文章的目录。
#├── data
#├── layouts
#├── static
#└── themes //是主题目录，可以去 themes.gohugo.io 下载喜欢的主题。

下载喜欢的主题:

themes.gohugo.io

也是下载到~/Downloads下的zip主题文件 解压 放在blog下的themes里

（主题包里一般会有demo模块 可复制里面的config.toml到blog下 更改配置文件以进行主题的个性化设置）

发布文章:

$ hugo new post/my-first-post.md

会在blog/content/post (post表示文章目录总)生成md后缀的文件

以后主要在md文件里进行markdown格式的文章编辑啦～（我用的是intellij idea编辑器）

本地浏览器预览:

$ hugo server -D //or
$ hugo server --theme=hugo-theme-zozo-master //--theme指定你的主题
#Web Server is available at http://localhost:1313/ (bind address 127.0.0.1)

打开http://localhost:1313/ 就可以在本地打开你的blog了

3.把本地hugo部署到github上

github新建一个项目:

  • new respository 新建一个站点 name 写 自己定义.github.io 确定

把blog构建为一个静态包

打开终端

 $ hugo --theme=hugo-theme-zozo-master --baseUrl="https://XXX.github.io/" //--baseUrl是刚在github上新建项目的name *

以上指令会在blog下生成一个public静态资源包

将public包上传到新建的respository里

终端进入blog/public

 $ git init
 $ git remote add origin https://github.com/XXX/XXX.github.io.git
 $ git add -A
 $ git commit -m "first commit"
 $ git push -u origin master

浏览器打开： ifhelen.github.io

至此 博客搭建完成！🎉
