git使用

把远程仓库clone到本地

$ git clone 仓库地址

本地修改后先add

$ git add .    //提交全部
$ git add XX XX   //提交某个

再提交

$ git commit -m  ""    //提交 -m 注释

再push

$ git push origin master //push到主干
